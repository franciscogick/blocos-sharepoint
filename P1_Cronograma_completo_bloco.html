<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Nesta atividade parte 1, o grupo vai elaborar o </span><strong><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">CRONOGRAMA DE ATIVIDADES</span></strong>, <span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;"> que consiste na distribui&ccedil;&atilde;o das atividades que fazem parte da execu&ccedil;&atilde;o do projeto cultural escolhido no per&iacute;odo de 10 semanas.</span></p>
<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">O projeto escolhido por voc&ecirc;s tem 10 atividades e voc&ecirc;s precisar&atilde;o distribu&iacute;-las no per&iacute;odo de 10 semanas.</span></p>
<p style="line-height: 1.35; text-align: justify; margin: 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Exemplo de atividades dos projetos: contrata&ccedil;&atilde;o da equipe, entrega do relat&oacute;rio de presta&ccedil;&atilde;o de contas, ensaios etc.</span></p>

<h2 style="font-size: 14pt; font-family: 'Open Sans','Arial',sans-serif; margin: 6pt 0 6pt 0; color: #000000;">COMO FAZER?</h2>
<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">A <b>primeira a&ccedil;&atilde;o</b> &eacute; ordenar as 10 atividades, ou seja, indicar qual ser&aacute; a primeira, a segunda e seguir at&eacute; a &uacute;ltima atividade.<br />Mas essa organiza&ccedil;&atilde;o tem uma l&oacute;gica. Por exemplo, n&atilde;o podemos apresentar uma pe&ccedil;a de teatro sem antes fazer ensaios, da mesma forma, n&atilde;o podemos lan&ccedil;ar um livro sem antes desenvolver a capa.</span></p>
<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">A <b>segunda a&ccedil;&atilde;o</b> &eacute; sobre a dura&ccedil;&atilde;o de cada atividade. O projeto escolhido por voc&ecirc;s ser&aacute; desenvolvido em 10 semanas e voc&ecirc;s definir&atilde;o quantas semanas cada atividade vai durar. </span></p>
<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;"><b>Aten&ccedil;&atilde;o:</b></span></p>
<ul>
<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Consultem as informa&ccedil;&otilde;es do projeto escolhido que est&atilde;o na ficha do projeto, voc&ecirc;s podem se basear nela na hora de distribuir as atividades no cronograma;</span></li>
<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Algumas atividades podem acontecer na mesma semana;</span></li>
<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">As atividades podem ter dura&ccedil;&otilde;es diferentes, ou seja, algumas podem acontecer em uma semana e outras durar v&aacute;rias semanas.</span></li>
</ul>
<p style="line-height: 1.35; text-align: justify; margin: 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Disponibilizamos um modelo do cronograma para ajud&aacute;-los.<br />Neste modelo voc&ecirc;s ver&atilde;o as 10 atividades do projeto escolhido, e tamb&eacute;m uma tabela com o espa&ccedil;o para registrar as duas a&ccedil;&otilde;es citadas: a sequ&ecirc;ncia de atividades e tamb&eacute;m a dura&ccedil;&atilde;o de cada atividade. </span></p>
</td>

<h2 style="font-size: 14pt; font-family: 'Open Sans','Arial',sans-serif; margin: 6pt 0 6pt 0; color: #000000;">COMO ENTREGAR?</h2>
<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">A entrega ser&aacute; feita aqui mesmo no F&oacute;rum de Mentoria.</span></p>
<ul>
<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">O grupo deve preencher a tabela que est&aacute; no link abaixo e enviar o arquivo, ou </span></li>
<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Imprimir a tabela e fazer manualmente com l&aacute;pis de cor e enviar o arquivo digitalizado (foto ou captura de tela), ou </span></li>
<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">Fazer a tabela a m&atilde;o no caderno, colorir, digitalizar (foto ou captura de tela) e enviar.</span></li>
</ul>
<p style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Open Sans','Arial',sans-serif; color: #000000;">A postagem da atividade s&oacute; precisa ser feita 1 vez, n&atilde;o h&aacute; necessidade de cada integrante do grupo enviar a atividade.</span></p>
