<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Na Parte 4, vamos realizar a <b>Presta&ccedil;&atilde;o de Contas</b> do projeto cultural.</span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">A <b>Presta&ccedil;&atilde;o de Contas</b> &eacute; o momento em que demonstramos que todo o nosso planejamento teve a execu&ccedil;&atilde;o correta, e assim comprovamos que os recursos foram bem empregados.</span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Nesta atividade, iremos realizar <b>3 a&ccedil;&otilde;es diferentes</b>:</span></p>

<ol>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Identificar na <b>Planilha de Relat&oacute;rio Financeiro</b> (dispon&iacute;vel abaixo), os itens com <span style="background: #ffffb1; border-radius: 3pt;">valor realizado</span> diferentes do <span style="background: #ffffb1; border-radius: 3pt;">valor previsto</span>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Realocar os valores na <b>Planilha de Relat&oacute;rio Financeiro</b>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Elaborar a <b>Justificativa</b> das altera&ccedil;&otilde;es realizadas.</span></li>

</ol>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Ent&atilde;o, vamos olhar primeiro para a <b>Planilha de Relat&oacute;rio Financeiro</b>, na qual o grupo encontrar&aacute; as atividades do projeto com suas respectivas descri&ccedil;&otilde;es, valores previstos e os valores realizados.</span></p>

</td>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Observem com aten&ccedil;&atilde;o as colunas <span style="background: #ffffb1; border-radius: 3pt;">Previs&atilde;o</span> e <span style="background: #ffffb1; border-radius: 3pt;">Realizado</span> de cada atividade. Aten&ccedil;&atilde;o, h&aacute; <b>2 valores</b> da planilha que est&atilde;o <b>diferentes</b>. Voc&ecirc;s, ent&atilde;o, v&atilde;o escolher outras duas atividades para distribuir esses valores que n&atilde;o foram utilizados. Esses valores devem ser informados, posteriormente, pelo grupo no documento de Justificativa.</span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"> Importante lembrar: </span></p>

<ol>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Conferir em cada atividade, o <b>VALOR DA PREVIS&Atilde;O</b> e <b>VALOR REALIZADO</b>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Identificar as atividades com <b>valores diferentes</b> (<span style="background: #ffffb1; border-radius: 3pt;">amarelo</span>); </span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Observar a &uacute;ltima linha chamada <b>Or&ccedil;amento Total</b>. Note que ao final da coluna <b>PREVIS&Atilde;O</b> temos um valor total e este <b>n&atilde;o</b> est&aacute; igual ao valor total da coluna <b>REALIZADO</b>. A diferen&ccedil;a desses valores est&aacute; representada no final da coluna <b>DIFEREN&Ccedil;A</b> (<span style="background: #ffc8ba; border-radius: 3pt;">em vermelho</span>). Ou seja, precisaremos realocar esse valor da diferen&ccedil;a;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Escolher <span style="background: #6967ff; color: white; border-radius: 3pt;">DUAS</span> atividades (exceto as que est&atilde;o em amarelo, pois os valores j&aacute; est&atilde;o diferentes);</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Distribuir os valores entre essas <span style="background: #6967ff; color: white; border-radius: 3pt;">DUAS</span> atividades;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Modificar <b>somente</b> a coluna do <b>valor realizado</b>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Certificar que agora os valores <b>total da previs&atilde;o</b> e <b>realizado</b> ficaram iguais.</span></li>

</ol>

<h2 style="font-size: 18pt; font-family: 'Nunito','Arial',sans-serif; margin-bottom: 6pt; color: #000000;">COMO REALIZAR A JUSTIFICATIVA</h2>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"> Ap&oacute;s zerar a diferen&ccedil;a dos valores na sua planilha, voc&ecirc;s v&atilde;o perceber que agora ser&atilde;o 4 valores diferentes do previsto. </span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"> S&atilde;o eles: das 2 atividades que j&aacute; estavam em amarelo e das outras 2 que voc&ecirc;s escolheram para distribuir o valor total da diferen&ccedil;a (que estava em vermelho na tabela). </span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"> No documento da Justificativa, &eacute; necess&aacute;rio detalhar essas 4 atividades, informando porque as altera&ccedil;&otilde;es foram necess&aacute;rias. </span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"> Usem a criatividade! </span></p>

<h2 style="font-size: 18pt; font-family: 'Nunito','Arial',sans-serif; margin-bottom: 6pt; color: #000000;">ENTREGA</h2>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"> Abaixo disponibilizamos a <b>Planilha de Relat&oacute;rio Financeiro</b> do projeto do grupo e um <b>Modelo de Justificativa</b>. <ul>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Imprimir a tabela e fazer manualmente com l&aacute;pis de cor e enviar o arquivo digitalizado (podendo ser foto/imagem); ou</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Ou fazer a tabela a m&atilde;o no caderno, digitalizar e enviar no F&oacute;rum de Mentoria.</span></li>

</ul>
