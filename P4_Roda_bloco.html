<td>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Na Parte 4, vamos realizar a <b>Prestação de Contas</b> do projeto cultural.</span>

</p>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">A <b>Prestação de Contas</b> é o momento em que demonstramos que todo o nosso planejamento teve a execução correta, e assim comprovamos que os recursos foram bem empregados.</span>

</p>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Nesta atividade, iremos realizar <b>3 ações diferentes</b>:</span>

</p>

<ol>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Identificar na <b>Planilha de Relatório Financeiro</b> (disponível abaixo), os itens com <span style="background: rgb(255, 255, 177); border-radius: 3pt;">valor realizado</span> diferentes do <span style="background: rgb(255, 255, 177); border-radius: 3pt;">valor previsto</span>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Realocar os valores na <b>Planilha de Relatório Financeiro</b>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Elaborar a <b>Justificativa</b> das alterações realizadas.</span></li>

</ol>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Então, vamos olhar primeiro para a <b>Planilha de Relatório Financeiro</b>, na qual o grupo encontrará as atividades do projeto com suas respectivas descrições, valores previstos e os valores realizados.</span>

</p>

</td>

</tr>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Observem com atenção as colunas <span style="background: rgb(255, 255, 177); border-radius: 3pt;">Previsão</span> e <span style="background: rgb(255, 255, 177); border-radius: 3pt;">Realizado</span> de cada atividade. Atenção, há <b>2 valores</b> da planilha que estão <b>diferentes</b>. Vocês, então, vão escolher outras duas atividades para distribuir esses valores que não foram utilizados. Esses valores devem ser informados, posteriormente, pelo grupo no documento de Justificativa.</span>

</p>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">

Importante lembrar:

</span>

</p>

<ol>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Conferir em cada atividade, o <b>VALOR DA PREVISÃO</b> e <b>VALOR REALIZADO</b>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Identificar as atividades com <b>valores diferentes</b> (<span style="background: rgb(255, 255, 177); border-radius: 3pt;">amarelo</span>);

</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Observar a última linha chamada <b>Orçamento Total</b>. Note que ao final da coluna <b>PREVISÃO</b> temos um valor total e este <b>não</b> está igual ao valor total da coluna <b>REALIZADO</b>. A diferença desses valores está representada no final da coluna <b>DIFERENÇA</b> (<span style="background: rgb(255, 200, 186); border-radius: 3pt;">em vermelho</span>). Ou seja, precisaremos realocar esse valor da diferença;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Escolher <span style="background: rgb(105, 103, 255); color: white; border-radius: 3pt;">DUAS</span> atividades (exceto as que estão em amarelo, pois os valores já estão diferentes);</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Distribuir os valores entre essas <span style="background: rgb(105, 103, 255); color: white; border-radius: 3pt;">DUAS</span> atividades;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Modificar <b>somente</b> a coluna do <b>valor realizado</b>;</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Certificar que agora os valores <b>total da previsão</b> e <b>realizado</b> ficaram iguais.</span></li>

</ol>

<h2 style="font-size: 18pt; font-family: 'Nunito','Arial',sans-serif; margin-bottom: 6pt; color: #000000;">COMO REALIZAR A JUSTIFICATIVA</h2>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">

Após zerar a diferença dos valores na sua planilha, vocês vão perceber que agora serão 4 valores diferentes do previsto.

</span>

</p>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">

São eles: das 2 atividades que já estavam em amarelo e das outras 2 que vocês escolheram para distribuir o valor total da diferença (que estava em vermelho na tabela).

</span>

</p>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">

No documento da Justificativa, é necessário detalhar essas 4 atividades, informando porque as alterações foram necessárias.

</span>

</p>

<p style="line-height: 1.5; text-align: justify;">

<span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">

Usem a criatividade!

</span>

Abaixo disponibilizamos a <b>Planilha de Relatório Financeiro</b> do projeto do grupo e um <b>Modelo de Justificativa</b>. A entrega será feita aqui mesmo no Fórum de Mentoria, da mesma forma que as Partes anteriores.

<p style="line-height: 1.5; text-align: justify;">

</span>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Imprimir a tabela e fazer manualmente com lápis de cor e enviar o arquivo digitalizado (podendo ser foto/imagem); ou</span></li>
