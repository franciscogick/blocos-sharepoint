            <p style="line-height: 1.35; text-align: justify;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">O <b>CHECKLIST DE ACOMPANHAMENTO</b> é uma planilha contendo as informações sobre as atividades do projeto, e ele serve para verificarmos a situação de cada uma delas. O CHECKLIST é feito após o cronograma que o grupo estipulou, com o objetivo de acompanhar se a produção está conseguindo cumprir as atividades do cronograma.</span></p>

            <p style="line-height: 1.35; text-align: justify;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Para os futuros projetos de vocês, como Produtores Culturais, lembrem-se que um CHECKLIST bem executado proporciona uma logística mais assertiva para realização de projetos culturais. Depois de montado o checklist, basta você ir “assinalando” as atividades executadas ao longo do projeto. </span></p>

            <p style="line-height: 1.35; text-align: justify;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">No arquivo de modelo (logo abaixo), vocês encontrarão 10 linhas para colocar as atividades do projeto e as colunas com as informações sobre cada atividade. <br />Nas colunas da esquerda para à direita, vocês irão preencher:</span></p>

            <ul>

                <li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Atividades</b> -  aponta o nome das atividades conforme o cronograma.                 <li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Semana(s) prevista(s) no cronograma</b> - indica a(s) semana(s) escolhida(s) pelo grupo na Atividade-Parte 1, o Cronograma;</span></li>

                <li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Situação</b> - informa se a atividade foi realizada ou ainda está em andamento. Para identificar, <b>marque um X</b> a coluna <u>Concluído</u> (para realizado) ou <u>Não Concluído</u> (para atividade em andamento); perceba que a linha da atividade vai mudar de cor dependendo de onde o X for marcado (verde para concluído, amarelo para não concluído);

                </span></li>

                <li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Responsáveis</b> - identifica a pessoa da equipe que é responsável pela atividade. Essa informação é obtida na ficha do projeto (logo abaixo);</span></li>

                <li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Observações</b> - espaço para incluir o motivo da não conclusão da atividade. Pode servir também para qualquer anotação que o grupo considere necessária.</span></li>

            </ul>
