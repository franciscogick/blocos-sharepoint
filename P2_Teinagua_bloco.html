<p style="line-height: 1.35; text-align: justify;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">O <b>CHECKLIST DE ACOMPANHAMENTO</b> &eacute; uma planilha contendo as informa&ccedil;&otilde;es sobre as atividades do projeto, e ele serve para verificarmos a situa&ccedil;&atilde;o de cada uma delas. O CHECKLIST &eacute; feito ap&oacute;s o cronograma que o grupo estipulou, com o objetivo de acompanhar se a produ&ccedil;&atilde;o est&aacute; conseguindo cumprir as atividades do cronograma.</span></p>

<p style="line-height: 1.35; text-align: justify;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Para os futuros projetos de voc&ecirc;s, como Produtores Culturais, lembrem-se que um CHECKLIST bem executado proporciona uma log&iacute;stica mais assertiva para realiza&ccedil;&atilde;o de projetos culturais. Depois de montado o checklist, basta voc&ecirc; ir &ldquo;assinalando&rdquo; as atividades executadas ao longo do projeto. </span></p>

<td style="background-color: #fdffcc; margin-bottom: 10pt; padding: 10pt; border-radius: 8pt;">

<p style="line-height: 1.35; text-align: justify;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">No arquivo de modelo (logo abaixo), voc&ecirc;s encontrar&atilde;o 10 linhas para colocar as atividades do projeto e as colunas com as informa&ccedil;&otilde;es sobre cada atividade. <br />Nas colunas da esquerda para &agrave; direita, voc&ecirc;s ir&atilde;o preencher:</span></p>

<ul>

<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Atividades</b> - aponta o nome das atividades conforme o cronograma. Lembrem-se de adicionar todas as atividades que foram inclu&iacute;das no cronograma;</span></li>

<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Semana(s) prevista(s) no cronograma</b> - indica a(s) semana(s) escolhida(s) pelo grupo na Atividade-Parte 1, o Cronograma;</span></li>

<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Situa&ccedil;&atilde;o</b> - informa se a atividade foi realizada ou ainda est&aacute; em andamento. Para identificar, <b>marque um X</b> a coluna <u>Conclu&iacute;do</u> (para realizado) ou <u>N&atilde;o Conclu&iacute;do</u> (para atividade em andamento); perceba que a linha da atividade vai mudar de cor dependendo de onde o X for marcado (verde para conclu&iacute;do, amarelo para n&atilde;o conclu&iacute;do); </span></li>

<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Respons&aacute;veis</b> - identifica a pessoa da equipe que &eacute; respons&aacute;vel pela atividade. Essa informa&ccedil;&atilde;o &eacute; obtida na ficha do projeto (logo abaixo);</span></li>

<li style="line-height: 1.35; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 11pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>Observa&ccedil;&otilde;es</b> - espa&ccedil;o para incluir o motivo da n&atilde;o conclus&atilde;o da atividade. Pode servir tamb&eacute;m para qualquer anota&ccedil;&atilde;o que o grupo considere necess&aacute;ria.</span></li>

</ul>
