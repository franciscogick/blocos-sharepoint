<h3 style="line-height: 1.5; text-align: justify;"><span style="font-size: 18pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Pe&ccedil;a de divulga&ccedil;&atilde;o</span></h3>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Na cria&ccedil;&atilde;o da pe&ccedil;a de divulga&ccedil;&atilde;o, a regra &eacute;: soltar a imagina&ccedil;&atilde;o! O grupo deve criar uma pe&ccedil;a de divulga&ccedil;&atilde;o, podendo ser:</span></p>

<ul>

<li style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">uma <b>imagem est&aacute;tica</b> (como um cartaz ou um card para redes sociais etc.);</span></li>

<li style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">um <b>v&iacute;deo</b> (pode ser no formato do youtube, do tiktok, dos stories do instagram etc.);</span></li>

<li style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">ou um <b>&aacute;udio</b> (pode ser no formato de spot de r&aacute;dio, &aacute;udio para enviar por whatsapp etc.). </span></li>

</ul>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Voc&ecirc;s podem usar qualquer ferramenta para essa cria&ccedil;&atilde;o, pode ser um formato digital, pode ser feito &agrave; m&atilde;o, o que for melhor para o grupo!</span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">A ideia &eacute; que a pe&ccedil;a de divulga&ccedil;&atilde;o dialogue com o p&uacute;blico-alvo que se pretende atingir.</span></p>

<p style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">As &uacute;nicas exig&ecirc;ncias s&atilde;o que voc&ecirc;s insiram na sua pe&ccedil;a de divulga&ccedil;&atilde;o o <b>Servi&ccedil;o</b> (dispon&iacute;vel na ficha do projeto) e a <b>barra de logos</b> do projeto (disponibilizada mais abaixo).</span></p>

<ul>

<li style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Nas <b>imagens est&aacute;ticas</b>, a barra de logos deve estar na parte de baixo e centralizada;</span></li>

<li style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Nos <b>v&iacute;deos</b>, a barra de logos deve aparecer no final;</span></li>

<li style="line-height: 1.5; text-align: justify;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Nos <b>&aacute;udios</b>, a barra de logos deve ser <i>narrada (falada) no final</i>; </span></li>

</ul>

<h2 style="font-size: 18pt; font-family: 'Nunito','Arial',sans-serif; margin-bottom: 6pt; color: #000000;">ENTREGA</h2>

<p style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">O grupo deve criar sua pe&ccedil;a de divulga&ccedil;&atilde;o e elaborar o Release conforme as orienta&ccedil;&otilde;es acima. Depois, deve fazer uma postagem com <b>os dois arquivos</b>, a Pe&ccedil;a de Divulga&ccedil;&atilde;o e o Release. </span></p>

<p style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Abaixo temos algumas possibilidades de entrega, escolham o que for mais confort&aacute;vel para voc&ecirc;s:</span></p>

<ul>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Preencher o modelo de Release que est&aacute; no link abaixo, ou</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Imprimir o modelo de Release e fazer manualmente com l&aacute;pis de cor e enviar o arquivo digitalizado (foto ou <abbr title="Usando a tecla PrintScreen no computador ou home (liga/desliga) + volume no telefone.">captura de tela</abbr>), ou</span></li>

<li style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Fazer o modelo de Release &agrave; m&atilde;o no caderno e digitalizar e enviar no f&oacute;rum de mentoria.</span></li>

</ul>

<p style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;">Os arquivos de v&iacute;deo e &aacute;udio podem ser anexados da mesma forma que os demais arquivos.</span></p>

<p style="line-height: 1.5; text-align: justify; margin: 0 0 3pt 0;"><span style="font-size: 12pt; font-family: 'Nunito','Arial',sans-serif; color: #000000;"><b>A entrega ser&aacute; feita aqui mesmo no F&oacute;rum de Mentoria</b>, da mesma forma que as Partes anteriores.
